What it does

Sales summary

Revenue, order count, units sold, and average order value at a glance.

Top products

Ranks your catalog by revenue so you know what's actually driving sales.

Revenue trend

A daily revenue sparkline rendered right in the terminal — no charts library.

Low-stock alerts

Flags products at or below your threshold so you reorder before you sell out.

About the project

Order Insights grew out of running a real e-commerce store, where keeping track of what's selling and what's running low usually means wrestling with a spreadsheet. This tool reads a standard order export and answers those questions in a second, straight from the command line.

It's written in pure Python with no external dependencies, ships with unit tests and sample data, and is packaged so it installs as a single command. Built and maintained by Karolis Braciulis.
